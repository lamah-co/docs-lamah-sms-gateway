## Welcome to Lamah SMS API

The Lamah SMS API provides a powerful and reliable platform for sending SMS messages, managing OTP verification, and handling communication workflows. Our API is designed to be simple, fast, and scalable for businesses of all sizes.

**Base URL**: `https://sms.lamah.com/api`
