## Get Your API Token

Before you can start sending SMS messages, you need to obtain your API token from the Lamah dashboard.

<Steps>
    <Step title="Create an Account">
        go to [https://geteshaar.com](https://geteshaar.com) and create an account.
    </Step>
  <Step title="Visit the Dashboard">
    Go to [https://sms.lamah.com](https://sms.lamah.com) and log into your account.
  </Step>
  <Step title="Navigate to Projects">
    In your dashboard, navigate to the **Projects** section.
  </Step>
  <Step title="Create or Select a Project">
    Create a new project or select an existing one.
  </Step>
  <Step title="Copy Your API Token">
    Locate and copy the **API token** provided in the project details.
  </Step>
    <Step title="Add Your Ip to whitelist">
        in project tab navigate to ip whitelist and add your ip address.
  </Step>
</Steps>

## Send Your First SMS

Now let's send your first SMS message using the Lamah API.

<CodeGroup>

```bash cURL
curl --request POST \
  "https://sms.lamah.com/api/sms/messages" \
  --header "Authorization: Bearer YOUR_API_TOKEN" \
  --header "Content-Type: application/json" \
  --data '{
    "message": "Hello from Lamah SMS API!",
    "sender": "Lamah",
    "payment_type": "wallet",
    "receiver": "00218911234567"
  }'
```

```javascript JavaScript
const response = await fetch('https://sms.lamah.com/api/sms/messages', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_TOKEN',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    message: 'Hello from Lamah SMS API!',
    sender: 'Lamah',
    payment_type: 'wallet',
    receiver: '00218911234567'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

url = "https://sms.lamah.com/api/sms/messages"
headers = {
    "Authorization": "Bearer YOUR_API_TOKEN",
    "Content-Type": "application/json"
}
data = {
    "message": "Hello from Lamah SMS API!",
    "sender": "Lamah",
    "payment_type": "wallet",
    "receiver": "00218911234567"
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```

```php PHP
<?php
$curl = curl_init();

curl_setopt_array($curl, array(
  CURLOPT_URL => 'https://sms.lamah.com/api/sms/messages',
  CURLOPT_RETURNTRANSFER => true,
  CURLOPT_CUSTOMREQUEST => 'POST',
  CURLOPT_POSTFIELDS => json_encode([
    'message' => 'Hello from Lamah SMS API!',
    'sender' => 'Lamah',
    'payment_type' => 'wallet',
    'receiver' => '00218911234567'
  ]),
  CURLOPT_HTTPHEADER => array(
    'Authorization: Bearer YOUR_API_TOKEN',
    'Content-Type: application/json'
  ),
));

$response = curl_exec($curl);
curl_close($curl);

echo $response;
?>
```

</CodeGroup>

## Understanding the Response

A successful SMS send will return a response like this:

```json
{
  "message_id": "msg_123456789",
  "status": "sent",
  "cost": 0.05,
  "balance_remaining": 99.95
}
```
